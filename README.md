Análisis de Ventas Minoristas y Rentabilidad

## Resumen del Proyecto
Este proyecto analiza un conjunto de datos de ventas minoristas para comprender el rendimiento de las ventas, la rentabilidad y las posibles oportunidades de mejora del negocio. El análisis se realizó con Python en Google Colab con bibliotecas como Pandas y Matplotlib.

## Objetivos
* Analizar el rendimiento total de las ventas y las ganancias
* Identificar patrones de ventas por región y categoría de producto
* Detectar productos que generan pérdidas significativas
* Explorar la relación entre la cantidad vendida y la rentabilidad

## KPI Clave
* Ventas Totales
* Beneficio Total
* Margen de Beneficio Promedio
* Porcentaje de Pedidos con Pérdidas

## Visualizaciones
El proyecto incluye varias visualizaciones para respaldar la información empresarial, como:

* Ventas por región
* Ventas por categoría de producto
* Principales productos que generan pérdidas
* Relación entre la cantidad vendida y las ganancias

## Herramientas Utilizadas
* Python
* Pandas
* Matplotlib
* Google Colab

## Conjunto de Datos
Conjunto de datos de ventas minoristas que contiene información sobre pedidos, productos, regiones, ventas y ganancias. 
Soure: Kaggle

## Insights
El análisis destaca las tendencias clave de ventas, identifica los productos que generan pérdidas y proporciona información que puede respaldar las estrategias de precios, la evaluación de costos y la toma de decisiones empresariales.
